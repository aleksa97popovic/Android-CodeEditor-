# Daffy Duck Full Stack UI
Jednostavan frontend demo sa login, registracijom i dashboard tabelom korisnika.

## Funkcionalnosti
- Login i registracija (frontend-only)
- Dashboard sa tabelom korisnika
- Brisanje korisnika iz tabele
- Lepa vizuelna tema sa Daffy Duck-om

## Kako pokrenuti
1. Klonirajte repo
2. Otvorite `index.html` u browseru
